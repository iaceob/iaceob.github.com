What is enoa?

enoa is a standard library for other framework. like json can support Gson, Jackson, Fastjson and more.

Other items listed below:

  • json
    Gson Jackson Fastjson
  • database
    Activerecord BeetlSQL Mybatis Trydb
    C3P0 Druid Hikaricp
  • template
    Freemark Velocity Beetl enjoy
  • logger
    JdkLog Slf4j Logback Log4j Log4j2
  • serialization
    Jdk FST Hessian Kryo
  • webserver
    NanoHTTPD Jetty Tomcat Netty FastCGI
  • And more...

enoa provides a unified api, as for what you choose.

Of course, enoa also has its own content, like Yosart, Toolkit, Trydb, Http, Eml and more.

Why use enoa?

enoa have a consistent interface, no matter what framework you use. and enoa is very easy. a small amount of code implements a lot of features.

How to use?

enoa now version is 0.0, you can choose your language to read wiki

    Not have your language wiki? help enoa traslate wiki to your language. enoa wiki written by gitbook.

    Show code case

    Http

    Http.request(HttpMethod.POST, EoUrl.with("https://httpbin.org").subpath("post"))
  .raw("{\"name\": \"yosart\"}", "application/json")
  .para("arg0", 1)
  .para("arg1", true)
  .emit();

    Repeater

    Repeater.createServer(new NanoHTTPDProvider())
  .listen(9100);

    Yosart

    Yosart.createServer()
  .handle(Method.GET, "/", (req) -> Resp.with(req).render("Hello world").end())
  .listen();
