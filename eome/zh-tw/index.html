enoa 是什麼?

enoa 是一個針對於其他框架的標準庫, 比如 json 可以支援 Gson, Jackson, Fastjson 以及更多

其他部分欄目的支援可見下方:

  • json
    Gson Jackson Fastjson
  • database
    Activerecord BeetlSQL Mybatis Trydb
    C3P0 Druid Hikaricp
  • template
    Freemark Velocity Beetl enjoy
  • logger
    JdkLog Slf4j Logback Log4j Log4j2
  • serialization
    Jdk FST Hessian Kryo
  • webserver
    NanoHTTPD Jetty Tomcat Netty FastCGI
  • And more...

enoa 針對你所使用的框架, 提供一致的 api

當然, enoa 也有自創的內容, 例如 Yosart, Toolkit, Trydb, Http, Eml and more.

為什麼使用 enoa?

enoa 無論你使用什麼框架, 都可以提供一致的介面. 此外, enoa 的使用非常簡單, 使用最少量的程式碼實現更多的功能.

如何使用?

enoa 當前版本是 0.0, 你可以選擇你的語言瀏覽使用者指南

    沒有你的語言文件? 幫助 enoa 將文件翻譯成你的語言. enoa 使用者指南使用 gitbook 書寫.

    程式碼展示

    Http

    Http.request(HttpMethod.POST, EoUrl.with("https://httpbin.org").subpath("post"))
  .raw("{\"name\": \"yosart\"}", "application/json")
  .para("arg0", 1)
  .para("arg1", true)
  .emit();

    Repeater

    Repeater.createServer(new NanoHTTPDProvider())
  .listen(9100);

    Yosart

    Yosart.createServer()
  .handle(Method.GET, "/", (req) -> Resp.with(req).render("Hello world").end())
  .listen();
