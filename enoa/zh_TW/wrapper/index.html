包裝器

包裝器

包裝器是指所有第三方 框架/服務 包裝器, 事實上 enoa 當然不可能將所有框架都重寫一邊, 當然也是接入第三方框架來進行完善. 包裝器可以說是 enoa 的基礎, 後續高級功能如果需要都會從包裝器中選擇依賴. 但是, 所有包裝器都並非強制依賴完全可以脫離 enoa 體系用於任何系統中.

現有的包裝器包括

  • 配置加載器
  • 資料庫
  • HTTP 客戶端
  • JSON
  • 日誌

  • NOSQL

    • Redis

  • 模板

  • 工具庫
  • XML

目前並非所有包裝器都開發完成, 具體情況見內文.
