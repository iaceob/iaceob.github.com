JSON

JSON

Json 包裝器, 封裝提供了多種 Json 框架輸出, 包括

  • EnoaJson
  • FastJson
  • Gson
  • Jackson

其中 EnoaJson 是默認實現, 無需任何依賴, 直接使用.

提供者

EnoaJson

默認實現, 無需配置, 支持 Json 序列化, 但無法進行反序列化操作, 如果需要反序列化操作請選擇其他實現.

EnoaJsonMgr.me().defJsonFactory(new EoJsonProvider());

EnoaJson 解析來自於 JFinal 工程中的 Json 工具. 並剔除了 JFinal 的一些依賴.

FastJson

EnoaJsonMgr.me().defJsonFactory(new FastjsonProvider());

Gson

EnoaJsonMgr.me().defJsonFactory(new GsonProvider());

Jackson

EnoaJsonMgr.me().defJsonFactory(new JacksonProvider());

使用

通常一個工程中, 只會使用一個 Json 工具, 因此在 EnoaJsonMgr 中未提供多個實例的支持, 但同樣的, 也是可以刨除 EnoaJsonMgr 直接調用提供者即可支持不同的實例.

EnoaJson jackson = new JacksonProvider().json();
String json = jackson.toJson(obj);

JsonKit

關於 Json 的使用, 提供了一個工具類 io.enoa.json.kit.JsonKit 該類提供 json 操作的靜態方法, 與 EnoaJson 一致, JsonKit 選擇的提供者就是 EnoaJsonMgr 設定的提供者.

String json = JsonKit.toJson(obj);

開發

Json 模塊同樣支持自定義提供者.

與 Json 提供者相關的接口

  • EoJsonFactory
  • EnoaJson

EoJsonFactory json 的提供者, 要求返回 EnoaJson 該提供者的具體實現.

調用關係:

EnoaJson = EoJsonFactory.json();
