JSON

JSON 插件

使用

Json 插件, 將包裝器中 Json 組件已插件形式接入到 Yosart 中, 並且插件中提供了 Json 渲染擴展.

使用方式:

Yosart.createServer()
  .plugin(new JsonPlugin(new EoJsonProvider()))
  .listen();

JsonPlugin 默認使用的提供者是 EoJsonProvider 如果需要其他的提供者, 創建 JsonPlugin 時在構造參數中傳入.

另 Json 插件中提供了 Json 渲染擴展, 使用方式

Yosart.createServer()
  .ext(new JsonRenderExt())
  .listen()

POM

<dependency>
  <groupId>io.enoa</groupId>
  <artifactId>yosart-plugin-json</artifactId>
</dependency>
