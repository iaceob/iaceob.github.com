路由

路由擴展

本文是依照默認路由 EyRouterExt 進行說明, 並不適用與其他路由擴展,

路由擴展實現自 YmRouterExt, 以下是默認路由的支持情況.

COLUMN SUPPORT DESCRIPTION
YoAction YES
YoControl YES
方法參數解析 NO 不支持解析, 但是允許方法參數, 會有警告
方法返回值解析 NO 不支持解析, 但是允許有返回值, 不會做任何處理, 會有警告
註解 NO 無任何註解

默認路由標準 Control 以及 Action 寫法.

Control

public class IndexControl extends YoControl<IndexControl> {
  public void index() {
    super.render("index");
  }

  @Action(method = Method.GET, value = "/action_test")
  public void actionTest() {
    super.render("action test");
  }
}

Control 需要繼承 YoControl, 便於獲取 Request 相關之, 以及 Render 系列方法維持整個生命週期是相同的 Resp

Action

new YoAction() {
  @Override
  public Response handle(YoRequest request) {
    return Resp.with(request).render("action").end();
  }
}

或

(req) -> Resp.with(req).attr("req", req.data()).renderTemplate("info").end()

支持加載的其他擴展

  • YmBeforeActionExt

    將會每個請求後, 在執行 Action/ Control 之前調用

  • YmExceptionExt

    將會在 Action/Control 執行拋出異常的情況
