工具庫

工具庫

工具庫可以說是衆多 enoa 專案的基石, 很多其他專案都引用了工具庫, toolkit 提供了很多工具的封裝, 字符處理, 集合處理, MAP 等等.

本文不對所有工具的方法代碼去單獨分析, 這樣就太長了, 這裏僅僅列出相應的類功效, 以及常用方法.

toolkit 一個最基本的理念是不依賴任何第三方包, 如果有依賴則默認不提供, 需要在引入. 因此在 密碼, id 生成依賴的第三方都默認不提供依賴, prop 以及 env 的加載又單獨提供了一份, 而不使用 conf 加載器.

io.enoa.toolkit.alg.UnitConvKit

單位換算, BIT, BYTE, KB, MB 等單位之間的轉換. (並爲全部完成)

io.enoa.toolkit.collection.CollectionKit

集合工具類, 提供集合以及 Map 空條件判斷, 空數組生成. 集合清除.

io.enoa.toolkit.convert.ConvertKit

類型轉換工具, 將某個類型數據轉換爲其他類型

io.enoa.toolkit.digest.Base64Kit

Base64 工具類

io.enoa.toolkit.digest.DigestKit

Hash 算法工具類

io.enoa.toolkit.digest.PasswdKit

密碼 加密 驗證, 加密驗證算法使用的是 BCrypt 算法, 因此在使用時需要引入 BCrypt 包, 默認不會引用

<dependency>
  <groupId>org.mindrot</groupId>
  <artifactId>jbcrypt</artifactId>
  <version>0.4</version>
</dependency>

io.enoa.toolkit.eo.tip.EnoaTipKit

該工具類是提供給 enoa 使用的, 但是外部使用也是可以, 該類的作用是獲取工程多語言的配置文件, 調用 message 方法將默認根據使用者系統的默認語言調用相應的資源消息. EnoaTipKit 加載配置文件的路徑是 classpath 中的 /eocnf/*.eo.properties

資源文件的命名需遵守規則

somename_en_US.eo.properties
somename_default.eo.properties

*_default.eo.perties 是在無法加載到當前系統語言後的默認加載項.

如果都不存在, 則返回不存在 key 的內容.

enoa 的異常消息顯示就是使用該類.

io.enoa.toolkit.file.FileKit

文件操作類

io.enoa.toolkit.id.snowflake.SnowflakeKit

Twitter snowflake id 生成算法, 依賴 snowflake, 默認不攜帶

<dependency>
  <groupId>xyz.downgoon</groupId>
  <artifactId>snowflake</artifactId>
  <version>1.0.0</version>
</dependency>

io.enoa.toolkit.image.ImageKit

圖片操作類

io.enoa.toolkit.map.Kv

Map 封裝, 提供鏈式調用

io.enoa.toolkit.map.Okv

與 Kv 一致, 只是繼承的 Map 是 LinkedHashMap , 一個有序的 Map.

io.enoa.toolkit.mark

標記接口, 常用於枚舉實現, 統一枚舉方法

io.enoa.toolkit.prop

該包下的類都和配置文件的加載有關, 該包下共有三個類

  • Prop
  • PropKit
  • Propx

Prop 用於加載 classpath 中的配置文件, PropKit 是 Prop 的靜態方法, 默認使用的最後一個加載的文件, 切換文件使用 use 方法, Propx 則沒有限制, 會去查找所有以加載的配置文件找到第一個匹配的就返回.

io.enoa.toolkit.ret

通用結果返回接口, 可用於方法返回.

io.enoa.toolkit.str.StrKit

字符串操作類

io.enoa.toolkit.stream.StreamKit

流操作工具

io.enoa.toolkit.sys

該包下的所有類均是系統操作工具類, 包括下列類

  • EnvKit

    讀取 System.getenv() 以及 System.getProperties() 配置

  • ExceptionKit

    異常工具

  • JavaKeyword

    Java 關鍵字

  • OSKit

    系統工具類, 獲取系統相關信息

  • PathKit

    路徑工具, 獲取路徑信息

  • ReflectKit

    反射工具

io.enoa.toolkit.thr

enoa 異常, 目前有兩個異常

  • EnoaException
  • EoException

EnoaException 繼承自 Exception 需要進行捕獲, 而 EoException 繼承自 RuntimeException 無需捕獲, 並且 message 提供 MessageFormat 進行格式化.

io.enoa.toolkit.EoConst

enoa 常量.
