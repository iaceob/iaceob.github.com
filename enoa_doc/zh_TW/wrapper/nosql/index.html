NOSQL

非關係型資料庫

非關係型資料庫有很多中, 常見的包括 Redis, MongoDB 等, 存在着非常大的差異, 不像關係型資料庫, 統一支持 SQL 查詢, 因此可以一起整合, 而非關係型資料庫只能進行單個封裝.

通常非關係型資料庫都會提供 cli 工具, enoa 的封裝將會完全依照 cli 提供的命令進行封裝.

目前 Redis 將會在後續版本中支持.
