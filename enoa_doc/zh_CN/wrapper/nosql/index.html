NOSQL

非关系型数据库

非关系型数据库有很多中, 常见的包括 Redis, MongoDB 等, 存在着非常大的差异, 不像关系型数据库, 统一支持 SQL 查找, 因此可以一起集成, 而非关系型数据库只能进行单个封装.

通常非关系型数据库都会提供 cli 工具, enoa 的封装将会完全依照 cli 提供的命令进行封装.

目前 Redis 将会在后续版本中支持.
