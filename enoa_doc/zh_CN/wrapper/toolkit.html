工具库

工具库

工具库可以说是众多 enoa 项目的基石, 很多其他项目都引用了工具库, toolkit 提供了很多任务具的封装, 字符处理, 集合处理, MAP 等等.

本文不对所有工具的方法代码去单独分析, 这样就太长了, 这里仅仅列出相应的类功效, 以及常用方法.

toolkit 一个最基本的理念是不依赖任何第三方包, 如果有依赖则默认不提供, 需要在引入. 因此在 密码, id 生成依赖的第三方都默认不提供依赖, prop 以及 env 的加载又单独提供了一份, 而不使用 conf 加载器.

io.enoa.toolkit.alg.UnitConvKit

单位换算, BIT, BYTE, KB, MB 等单位之间的转换. (并为全部完成)

io.enoa.toolkit.collection.CollectionKit

集合工具类, 提供集合以及 Map 空条件判断, 空数组生成. 集合清除.

io.enoa.toolkit.convert.ConvertKit

类型转换工具, 将某个类型数据转换为其他类型

io.enoa.toolkit.digest.Base64Kit

Base64 工具类

io.enoa.toolkit.digest.DigestKit

Hash 算法工具类

io.enoa.toolkit.digest.PasswdKit

密码 加密 验证, 加密验证算法使用的是 BCrypt 算法, 因此在使用时需要引入 BCrypt 包, 默认不会引用

<dependency>
  <groupId>org.mindrot</groupId>
  <artifactId>jbcrypt</artifactId>
  <version>0.4</version>
</dependency>

io.enoa.toolkit.eo.tip.EnoaTipKit

该工具类是提供给 enoa 使用的, 但是外部使用也是可以, 该类的作用是获取工程多语言的配置文档, 调用 message 方法将默认根据用户系统的默认语言调用相应的资源消息. EnoaTipKit 加载配置文档的路径是 classpath 中的 /eocnf/*.eo.properties

资源文档的命名需遵守规则

somename_en_US.eo.properties
somename_default.eo.properties

*_default.eo.perties 是在无法加载到当前系统语言后的默认加载项.

如果都不存在, 则返回不存在 key 的内容.

enoa 的异常消息显示就是使用该类.

io.enoa.toolkit.file.FileKit

文档操作类

io.enoa.toolkit.id.snowflake.SnowflakeKit

Twitter snowflake id 生成算法, 依赖 snowflake, 默认不携带

<dependency>
  <groupId>xyz.downgoon</groupId>
  <artifactId>snowflake</artifactId>
  <version>1.0.0</version>
</dependency>

io.enoa.toolkit.image.ImageKit

图片操作类

io.enoa.toolkit.map.Kv

Map 封装, 提供链式调用

io.enoa.toolkit.map.Okv

与 Kv 一致, 只是继承的 Map 是 LinkedHashMap , 一个有序的 Map.

io.enoa.toolkit.mark

标记接口, 常用于枚举实现, 统一枚举方法

io.enoa.toolkit.prop

该包下的类都和配置文档的加载有关, 该包下共有三个类

  • Prop
  • PropKit
  • Propx

Prop 用于加载 classpath 中的配置文档, PropKit 是 Prop 的静态方法, 默认使用的最后一个加载的文档, 切换文档使用 use 方法, Propx 则没有限制, 会去查找所有以加载的配置文档找到第一个匹配的就返回.

io.enoa.toolkit.ret

通用结果返回接口, 可用于方法返回.

io.enoa.toolkit.str.StrKit

字符串操作类

io.enoa.toolkit.stream.StreamKit

流操作工具

io.enoa.toolkit.sys

该包下的所有类均是系统操作工具类, 包括下列类

  • EnvKit

    读取 System.getenv() 以及 System.getProperties() 配置

  • ExceptionKit

    异常工具

  • JavaKeyword

    Java 关键字

  • OSKit

    系统工具类, 获取系统相关信息

  • PathKit

    路径工具, 获取路径信息

  • ReflectKit

    反射工具

io.enoa.toolkit.thr

enoa 异常, 目前有两个异常

  • EnoaException
  • EoException

EnoaException 继承自 Exception 需要进行捕获, 而 EoException 继承自 RuntimeException 无需捕获, 并且 message 提供 MessageFormat 进行格式化.

io.enoa.toolkit.EoConst

enoa 常量.
