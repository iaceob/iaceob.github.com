包装器

包装器

包装器是指所有第三方 框架/服务 包装器, 事实上 enoa 当然不可能将所有框架都重写一边, 当然也是接入第三方框架来进行完善. 包装器可以说是 enoa 的基础, 后续高级功能如果需要都会从包装器中选择依赖. 但是, 所有包装器都并非强制依赖完全可以脱离 enoa 体系用于任何系统中.

现有的包装器包括

  • 配置加载器
  • 数据库
  • HTTP 客户端
  • JSON
  • 日志
  • NOSQL
    • Redis
  • 模板
  • 工具库
  • XML

目前并非所有包装器都开发完成, 具体情况见内文.
