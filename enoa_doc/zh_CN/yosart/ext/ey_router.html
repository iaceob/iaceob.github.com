路由

路由扩展

本文是依照默认路由 EyRouterExt 进行说明, 并不适用与其他路由扩展,

路由扩展实现自 YmRouterExt, 以下是默认路由的支持情况.

COLUMN SUPPORT DESCRIPTION
YoAction YES
YoControl YES
方法参数解析 NO 不支持解析, 但是允许方法参数, 会有警告
方法返回值解析 NO 不支持解析, 但是允许有返回值, 不会做任何处理, 会有警告
注解 NO 无任何注解

默认路由标准 Control 以及 Action 写法.

Control

public class IndexControl extends YoControl<IndexControl> {
  public void index() {
    super.render("index");
  }

  @Action(method = Method.GET, value = "/action_test")
  public void actionTest() {
    super.render("action test");
  }
}

Control 需要继承 YoControl, 便于获取 Request 相关之, 以及 Render 系列方法维持整个生命周期是相同的 Resp

Action

new YoAction() {
  @Override
  public Response handle(YoRequest request) {
    return Resp.with(request).render("action").end();
  }
}

或

(req) -> Resp.with(req).attr("req", req.data()).renderTemplate("info").end()

支持加载的其他扩展

  • YmBeforeActionExt

    将会每个请求后, 在运行 Action/ Control 之前调用

  • YmExceptionExt

    将会在 Action/Control 运行抛出异常的情况
