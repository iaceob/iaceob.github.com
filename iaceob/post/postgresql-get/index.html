PostgreSQL 新技能

通常数据库的模糊查询语句如下

select * from t_exp where column like '%xxx%' -- '%s' 's%'

PostgreSQL 有个进阶方案可以使用 ~ 代替

select * from t_exp where column ~ 'x';

这就相当于 like '%x%'

同时 经测试发现 ~'' 查出的是所有的数据, 当查询值存在的时候 比如 ~'x' 相当于 like '%x%'

新技能. 简单的模糊查询会方便很多
