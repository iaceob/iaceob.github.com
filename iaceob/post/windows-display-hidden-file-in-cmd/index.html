显示隐藏文件的cmd命令

1。解除对某磁盘某个文件夹的强行隐藏(示例)：

attrib +s +h +a c:boot.ini

2，解除对某磁盘全部文件夹的强行隐藏(示例)：

attrib -s -h c:boot.ini

以上示例中，d:为磁盘盘符；引号内为文件夹名称；相关命令参数：
+r 设置只读文件属性。
-r 清除只读文件属性。
+a 设置存档属性。
-a 清除存档属性。
+s 设置系统文件属性。
-s 清除系统文件属性。
+h 设置隐藏文件属性。
-h 清除隐藏文件属性。
/s 将attrib和任意命令行选项应用到当前目录及其所有子目录中的匹配文件。
/d 将attrib和任意命令行选项应用到目录。
/? 在命令提示符下显示帮助。
