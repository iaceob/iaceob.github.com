Sublimetext 2 修改编辑器字体

Sublimetext 2 是一款非常好的编辑器，使用之后就会感觉到它的强悍。

在使用的时候想将显示的字体修改成自己想要的字体，设置如下:

打开 Preferences -> File Settings - User 在显示的窗口中会看到 { } 这样的内容，默认没有配置，修改如下语句

{
"font_face": "Verdana",
"font_size": 10
}

编辑器字体就会修改成设定的字体。
