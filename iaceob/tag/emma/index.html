ubuntu emma 中文乱码

emma 是 ubuntu 中一个mysql图形管理工具

默认emma使用的编码是latin1那么数据中如果有中文的话查询出来会是乱码。

设置emma的编码为utf8查询出来的中文就会正常

执行如下命令：

~$ sudo sublime ~/.emma/emmarc

修改 db_encoding 值为 utf8

db_encoding=utf8

重新启动emma，在查询就会正常显示。

如果还是乱码 执行这条sql便会正常

set names utf8
