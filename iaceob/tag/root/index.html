MySQL 重设root密码

Stop Mysql // 停止mysql服务

服务不检测权限状态登录:mysqld_safe -skip-grant-tables

空密码方式使用root用户登录

$ mysql-uroot -p

更改Root密码:mysql> update mysql.user set password=Password(’newpassword’) where User=’root’;

mysql> flush privileges;
mysql> exit

重启Mysql
