ubuntu apache 开启 url rewriter

如果安装了rewrite模块的话直接输入命令

sudo a2enmod rewrite

即可开启url rewriter。

当然还要在apache的配置文件中做修改，否则就算开启了服务可还是不能使用

打开apache配置文件

sudo gedit /etc/apache2/sites-enabled/000-default

将 AllowOverride None 修改成 All~ All

AllowOverride All
